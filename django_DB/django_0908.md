### Static files

개발자가 보는 곳

변화가 적음

정적 파일

응답할 때 별도의 처리 없이 파일 내용을 그대로 보여주면 되는 파일

웹사이트의 경우 이미지, 자바스크립트, CSS와 같은 미리 준비된 추가파일을 제공해야함

##### STATIC_ROOT

collectstatic이 배포를 위해 정적 파일을 수집하는 디렉토리의 절대 경로

django 프로젝트에서 사용하는 모든 static file을 한곳에 모아넣는 경로
