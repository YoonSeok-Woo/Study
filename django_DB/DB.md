# DB

### 데이터베이스로 얻는 장점

* 데이터 중복 최소화
* 데이터 무결성(정확한 정보를 보장)
* 데이터 일관성
* 데이터 독립성(물리적/논리적)
* 데이터 표준화
* 데이터 보안 유지

### 관계형 데이터 베이스(RDB)

* Relational Database
* 키와 값들의 간단한 관계를 표 형태로 정리한 데이터베이스
* 관계형 모델에 기반
* 스키마(schema): 데이터베이스에서 자료의 구조, 표현방법, 관계등 전반적인 명세를 기술한 것.
* 테이블(table): 열(컬럼/필드)과 행(레코드/값) 의 모델을 사용해 조직된 데이터 요소들의 집합
* RDBMS: 관계형 데이터 베이스로 이루어진 데이터베이스를 관리하는 시스템
